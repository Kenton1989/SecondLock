# Functionality

插件功能与何同学的时间锁App相似，用于在浏览含有大量信息流，容易过度消耗时间的网页（如各种视频网站，社交平台）之前，先给自己设定时间限额，时间到了之后，自动阻止浏览。

各个网站默认不需要拦截，除了被加入到黑名单中的网站。毕竟有能力提供大量信息流供人消遣的网站在世界上其实占少数。
