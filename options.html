    Monitored Host

    Details about how the list works:
    • All the subdomain of host will be monitored.
      (e.g. if "youtube.com" is in the list, then "www.youtube.com", "music.youtube.com", etc. will be monitored at the same time)
    • Adding a super domain will eliminate all its subdomains from the list.
      (e.g. Suppose "www.youtube.com", "music.youtube.com" are in the list. If "youtube.com" is then added, both of them will be replaced by "youtube.com"
    • To add a new subdomain, you need to delete the parent domain first, and you must save the list before adding the subdomain.
      (e.g. Suppose "youtube.com" is in the list, and you want to add "www.youtube.com". You must delete "youtube.com" form the list and save it before adding "www.youtube.com" into the list.)
    • You can either hit ENTER or click "Add" button to add a new hostname after entering.
    • Only HTTP and HTTPS page is monitored. Other protocols like FTP are not monitored.
